== Description ==
This is an Online Reservation System, which gives customers the opportunity to choose the date and and the time of an appointment according to one's own preferences and the booking can now be done online.

The reservation system can be added to any website, it is not necessary to make a new one. It can be added as a new page or in a new space. Sites that use the online booking system were made ​​by Synergy-Consulting, by other agencies or the web site owner himself.

To learn more about our booking service please visit : www.my-reservation-system.com

**NOTE: You need to register on reservation site and then you will get a Authentication token which you will use to validate this Plugin. **

== Installation ==
Follow this steps to install MF2

1. Download the plugin into the **/wp-content/plugins/** folder
2. Activate MRS
